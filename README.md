# 🎮 GameXP – Gamified Loyalty Platform

GameXP is an interactive web-based platform that combines gaming with a reward system to enhance user engagement. Users can play games, earn XP points, unlock rewards, and compete on leaderboards — all within a single web interface.

---

## 🚀 Features

* 🎮 Multiple Playable Games (2048, Snake, Memory Flip, etc.)
* 🧠 XP / Points System using browser storage (LocalStorage)
* 🏆 Leaderboard system for ranking players
* 🎖️ Badge & Achievement system
* 🛍️ Reward redemption system
* 👤 User session persistence
* 📱 Fully responsive design (mobile + desktop)
* 🎨 Smooth UI with animations and interactive elements

---

## 🔥 Key Highlights

* Built as a **complete gamified ecosystem**, not just standalone games
* Implements a **persistent XP and reward system without a backend**
* Modular structure allowing **easy future backend integration**
* Demonstrates **real-world use of gamification in user engagement systems**

---

## ⚙️ How It Works

* XP is stored and managed using **LocalStorage**
* Each game updates XP based on user performance
* Leaderboard dynamically ranks users based on XP
* Rewards are unlocked and redeemed based on XP thresholds
* User session is persisted using browser storage
